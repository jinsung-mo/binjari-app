# 창원대학교 스마트 주차장 모니터링 서비스 - 빈자리 (Binjari)

창원대학교 컴퓨터공학과 캡스톤 디자인 프로젝트로 개발된 스마트 주차장 모니터링 시스템입니다. YOLOv8 딥러닝 모델을 활용하여 실시간으로 주차 공간의 점유 상태를 파악하고, 사용자에게 최적화된 주차 정보를 제공합니다.

## 1. 프로젝트 제안 배경

현재 교내 주차 환경은 다음과 같은 한계점을 가지고 있습니다.
- 실시간 정보 부재: 사전에 주차 가능 여부를 확인할 수 없어 모든 구역을 탐색해야 하는 번거로움 발생
- 비효율적 공간 운영: 특정 구역에만 차량이 밀집되고 외곽 주차장은 유휴 공간으로 남는 불균형 현상
- 환경 및 안전 문제: 주차 공간 탐색을 위한 공회전으로 인한 배기가스 배출 및 보행자 안전 위험 증가

본 프로젝트는 저비용 고효율의 모니터링 시스템을 구축하여 이러한 문제를 해결하고자 합니다.

## 2. 주요 기능

- 실시간 주차 현황 확인: 캠퍼스 맵 기반의 직관적인 UI를 통해 현재 빈자리 위치 확인
- 주차 점유 데이터 시각화: 시간대별 이용률 및 통계 데이터를 차트로 제공
- 관리자 제어 기능: 주차 구역 설정 및 시스템 구성 관리
- 멀티 플랫폼 지원: Flutter 기반으로 Android, iOS, Web, Desktop 환경 지원

## 3. 핵심 기술 및 알고리즘

### 차량 인식 및 점유 판단 (YOLOv8)
- 모델: Ultralytics의 YOLOv8 모델을 사용하여 실시간 객체 탐지 수행
- 점유 판별 알고리즘:
    - 겹침 비율 = (교차 영역 / 주차 공간 면적)
    - 최종 스코어 = 겹침 비율 X 객체 탐지 신뢰도
- 정밀도 향상 기법:
    - 시간적 필터링: 단일 프레임의 불안정성을 해결하기 위해 최신 프레임에 높은 가중치를 부여한 가중 평균 적용
    - 상태 전이 모델: 'empty'와 'occupied' 간 상태 변경 시 연속된 프레임(3~8프레임) 확인을 통해 일시적인 오탐지 방지

### 시스템 아키텍처
1. 데이터 취득: 스마트폰, CCTV 또는 웹캠을 통한 주차장 영상 촬영
2. 분석 서버 (Python/Flask): YOLOv8 모델을 통한 차량 검출 및 주차 여부 인식
3. 데이터 저장: 분석된 주차 위치 좌표 및 상태를 SQLite 데이터베이스에 저장
4. 클라이언트 (Flutter): RESTful API를 통해 데이터를 수신하여 실시간 현황 시각화

## 4. 기술 스택

- Frontend: Flutter (Dart)
- Backend: Python (Flask)
- Deep Learning: YOLOv8 (PyTorch)
- Database: SQLite
- Communication: REST API (CORS 지원)

## 5. 기대 효과

- 효율적인 시간 관리: 주차 소요 시간 단축을 통해 일상 속 생산성 향상
- 스트레스 감소: 빈자리 탐색으로 인한 심리적 불안감 및 짜증 해소
- 탄소 배출 저감: 탐색 주행 거리 단축을 통한 친환경 캠퍼스 조성
- 데이터 기반 관리: 주차 통계 분석을 통한 효율적인 시설 관리 정책 수립
